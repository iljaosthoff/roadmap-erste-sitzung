# Die erste Hypnosesitzung — Interaktive Roadmap

Eine interaktive Schritt-für-Schritt-Roadmap für die erste Hypnosesitzung. Von der Begrüssung bis zur Nachsorge — mit aufklappbaren Praxis-Tipps aus dem klinischen Alltag.

Erstellt von **Ilja Osthoff** — Anästhesist, Hypnosetherapeut und Gründer der Community [Die Hypnose Enthusiasten](https://www.skool.com/die-hypnose-enthusiasten).

---

## Inhalt

Die Roadmap führt durch 8 Schritte der ersten Hypnosesitzung:

1. Begrüssung & Beziehungsaufbau
2. Anamnese & Zielformulierung
3. Arbeitsauftrag & Erwartungsklärung
4. Hypnoseerklärung
5. Induktion
6. Intervention
7. Exduktion & Reorientierung
8. Nachgespräch & Integration

Jeder Schritt enthält einen aufklappbaren Praxis-Tipp aus dem klinischen Alltag.

---

## Technik

- Reines HTML/CSS/JS — keine Abhängigkeiten, kein Build-Prozess
- Einzelne `index.html` — direkt deploybar
- Responsive für Mobile und Desktop
- Google Fonts: Bebas Neue + Montserrat

## Deployment

**GitHub Pages:**
1. Repository forken oder klonen
2. In den Repository-Einstellungen unter *Pages* den Branch `main` und den Ordner `/ (root)` auswählen
3. Die Seite ist unter `https://[dein-username].github.io/[repo-name]` erreichbar

**Static.app:**
Den Ordner als ZIP hochladen — `index.html` wird automatisch erkannt.

**Beliebiger Webserver:**
Die `index.html` einfach in das Root-Verzeichnis legen.
